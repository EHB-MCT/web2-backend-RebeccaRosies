Persic_information

Get

  • Get / = root info page
  • Get /songs = get all the songs from the collection 'songs'
  • Get /favSongs = get all the songs that in the 'favSongs' collection (all the songs you favorited)

Post

  • Post /songs = post songs to the collection 'songs'
  • Post /favSongs = get all the songs that in the 'favSongs' collection (all the songs you favorited)

    • -> provide a body of: song = {"name": "songname", "artist" = "artistname", "genre" = "genrename", "rating" = rating}

PUT

  • Put /songs/:id = change a song that is in the 'songs' collection
  • Put /favSongs/:id = change a song that is in the 'favSongs' collection (all the songs you favorited)

    • -> provide a body of: song = {"name": "songname", "artist" = "artistname", "genre" = "genrename", "rating" = rating}

Delete

  • Delete /songs = delete all song that are in the 'songs' collection
  • Delete /Songs/:id = delete a song that is in the 'songs' collection
  • Delete /favSongs/:id = delete a song that is in the 'favSongs' collection (all the songs you favorited)
